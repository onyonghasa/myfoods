Usage Instructions

For Administrators:

  1. Use the "Recipe Management" menu to add/edit/delete food recipes.
  2. Fill out the food information form completely.
  3. Define the crafting recipe and required ingredients.
  4. Click "Update file datafoods.json" to save the changes.
  5. Upload the datafoods.json file to your server.
  6. You can place the file indexlitepreview.html on your host along with the datafoods.json file so players can view the food list.

For Players:

  1. Use the "Recipe Preview" menu to view all food recipes.
  2. Click on a food item to see details and the crafting recipe.
